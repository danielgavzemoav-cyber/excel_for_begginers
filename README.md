# # CSV Analysis Tool

A Python data analysis pipeline for CSV files.

## Live Demo
[Click here to try the app](https://excelforbegginers-q9uacgbudsmv5y6b8kzgbd.streamlit.app/)

## Features
- Automatic data cleaning and type detection
- Statistical analysis (RMS, amplitude, covariance, FFT)
- Curve fitting (polynomial, exponential, trigonometric)
- Machine learning (polynomial regression, anomaly detection)
- Interactive web interface built with Streamlit

## Files
- `cleaning.py` — data cleaning
- `float_analysis.py` — statistical analysis
- `fittings.py` — curve fitting
- `ML_Analysis.py` — machine learning
- `streamlit_app.py` — web interface
- `main.py` — CLI interface

## How to run locally
pip install -r requirements.txt
streamlit run streamlit_app.py
